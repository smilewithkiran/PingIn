📤
Smart Outreach Campaign NEW v2.4
Run fully automated LinkedIn outreach campaigns directly from PingIN. No new windows, no tab switching — PingIN navigates your existing LinkedIn tab, sends personalised messages, and tracks every action.
Navigates existing LinkedIn tab — fast, natural, and LinkedIn-safe
Campaign state persists — close popup, switch tabs, reopen and it continues where it left off
Zero duplicate sends — permanent URL tracking across all sessions
3 campaign types: Connect, Message, Visit
Personalised templates with {{first_name}}, {{company}}, {{role}}, {{location}}
GitHub-specific "Found your work on GitHub" template for developer outreach
Daily safety limit: up to 50 messages/day within LinkedIn's safe zone
Live campaign dashboard — sent, pending, failed — visible even after popup close

Developer Search NEW v2.6
Find developers where they actually live — GitHub and Stack Overflow. Verified by real code, commits and answers. No login needed. Completely free platforms.
GitHub Smart Search — builds: language:Python location:Bangalore followers:>=100 repos:>=20 — opens ranked by most active
Stack Overflow Top Experts — opens stackoverflow.com/tags/react/topusers — top 20 ranked experts for any skill tag
GitHub Multi-Skill Search (1-3 skills) — search by Skill 1 + Skill 2 + Skill 3. GitHub finds developers mentioning ALL entered skills in their profile bio. Results sorted by most followers first.
GitHub Profile Banner — PingIN floating panel appears on every github.com/username page automatically — Find LinkedIn, Add to List buttons
3-level LinkedIn fallback: direct link → name search → "Direct profile not found. Opened LinkedIn with matching skills — you might find someone even better."
⚡
4 New Productivity Features NEW v2.8
Search History, Candidate Notes, Boolean Builder and JD Library — four features that save hours every week.
⚡ Boolean Builder — add skill chips, choose OR/AND logic, click Apply → PingIN builds the Boolean string automatically. No Boolean experience needed.
🕐 Search History — last 15 searches saved automatically. Click any to instantly re-run it — fills all fields and searches. Never type the same search twice.
📝 Candidate Notes — private notes on every profile card. Preview shows under the name. "Called 12 Jan — expects 28L, follow up March." Persists forever.
📁 JD Library — save up to 10 job descriptions. Click any saved JD to load and auto-analyse. Works with JD Search — paste once, reuse forever.

AI Clone Search · JD Parsing · Predictive Sourcing
Three Claude AI-powered search modes. Available on Pro and Annual plans.
